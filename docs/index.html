webLISP Playground

webLISP is a nano Common-LISP implementation. Its main purpose is educational based. Implemented operations:
>, <, *, +, -, /, car, cdr, cons, cos, defun, defvar, dotimes, exp, if, let, let*, max, min, random, setq, sin, sqrt, terpri, write, zerop .

Input

         limit runtime to seconds

Output

(empty)
